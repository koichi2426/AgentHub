# domain

このディレクトリはインフラ層から見たドメイン関連の実装を置く場所です。

目的:
- ドメインエンティティの永続化に必要なインフラ固有の実装や、ORM マッピング定義をここに置きます。
- Repository 実装で使うエンティティ定義（例えば SQLAlchemy のモデル）や、ドメインと DB スキーマを結ぶアダプターを含めます。

例:
- `models.py`（SQLAlchemy のデータモデル）
- `mappers.py`（ドメインオブジェクト ↔ DB モデルの変換）

実装ノート:
- ドメインロジックそのものは `backend/domain` に置き、ここはあくまでインフラ寄りの表現（ORM や永続化のための補助）を置きます。
